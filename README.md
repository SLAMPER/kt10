Задание 3

Задание: Напишите обобщенный интерфейс IComparable<T>, который содержит метод int CompareTo(T y), который возвращает целое число, указывающее, как текущий объект сравнивается со вторым объектом типа T. Затем напишите структуры ComplexNumber и RationalNumber, которые представляют комплексные числа и рациональные числа соответственно и реализуют интерфейс IComparable<T> для своих типов.
